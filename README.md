# ISO 27001 Mini Gap Assessment

## Overview

This project demonstrates a simplified ISO/IEC 27001 readiness assessment performed for a fictional organisation.

The objective was to evaluate the organisation's current information security maturity, identify gaps against ISO 27001 expectations and create an improvement roadmap to support future implementation activities.

The project was designed to simulate a common Governance, Risk and Compliance (GRC) activity performed before an ISO 27001 implementation or certification project.

---

## Project Objectives

- Assess current information security maturity
- Review existing ISMS documentation
- Identify compliance and security gaps
- Evaluate readiness for ISO 27001 implementation
- Create an improvement roadmap
- Demonstrate practical GRC assessment skills

## Gaps Identified

### Security Awareness Training

Gap:
No formal awareness programme exists.

Recommendation:
Develop and deliver security awareness training.

Priority:
High

---

### Security Monitoring

Gap:
No documented monitoring process.

Recommendation:
Implement monitoring procedures and responsibilities.

Priority:
High

---

### Backup Management

Gap:
Backup procedures not documented.

Recommendation:
Develop and maintain backup procedures.

Priority:
High

---

### Incident Response

Gap:
No documented incident response process.

Recommendation:
Create and maintain an Incident Response Plan.

Priority:
High

---

## Readiness Assessment

Overall Readiness Level:

**Moderate**

The organisation has implemented several foundational ISO 27001 governance and risk management activities but requires additional operational controls and supporting procedures to achieve greater ISMS maturity.
